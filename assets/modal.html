Notes

Acaso merupakan aplikasi untuk mengambil value random dari sebuah array.


Metode

  • Manual - Normal, array ditulis manual oleh user, dimana setiap value dipisah dengan keyword "-". Array bersifat statis, tidak terpengaruh oleh value yang muncul. Contoh penulisan array : Kucing - Burung - Ikan.
  • Manual - Eliminasi, Penulisan array sama seperti metode-1. Array bersifat dinamis, value yang muncul akan dihapus dari array.
  • Auto - Normal, metode auto dikhususkan untuk array numerik, dimana array didapat dari inputan angka pertama dan terakhir. Contoh penulisan array : Awal = 1, Akhir = 5. Maka array yang didapat adalah 1,2,3,4,5. Array bersifat statis sama seperti metode-1.
  • Auto - Eliminasi, Penulisan array sama seperti metode-3, dan array bersifat dinamis sama seperti metode-2.

Fungsi Tombol

  • Lock & Unlock, untuk mengunci dan membuka array yang dimasukkan. (Nb: Pastikan format pengisian array sudah benar)
  • Play, untuk menjalankan program acaso. (Nb: Array harus dikunci terlebih dahulu sebelum dijalankan)
  • Stop, untuk menghentikan dan mendapatkan hasil dari program yang dijalankan.
  • Delete, untuk menghapus array yang dibuat.
